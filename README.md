## 📖 Overview

A comprehensive analysis of **medical health insurance data** covering cost prediction, risk factor identification, and demographic analysis. This project helps understand the key drivers of healthcare costs and builds predictive models for insurance premium estimation.

---

## 🎯 Analysis Goals

- Identify key risk factors that drive medical insurance costs
- Build regression models to predict insurance premiums
- Analyze demographic patterns in healthcare utilization
- Visualize cost distributions across different population segments

---

## 📊 Dataset Features

| Feature | Type | Description |
|---------|------|-------------|
| age | Numeric | Patient age |
| sex | Categorical | Gender |
| bmi | Numeric | Body Mass Index |
| children | Numeric | Number of dependents |
| smoker | Categorical | Smoking status |
| region | Categorical | Geographic region |
| charges | Target | Annual insurance charges |

---

## 📈 Key Insights

- Smokers incur 3-4x higher insurance charges than non-smokers
- BMI above 30 significantly increases medical costs
- Age has a strong positive correlation with insurance charges
- Regional differences account for ~5% of cost variation
